Getting Started as an Auditor

Learn how to effectively review and resolve disputes on Dodao as an auditor.

Overview​

As an auditor on Dodao, you can:

  • Review disputed tasks
  • Evaluate evidence from both parties
  • Make fair and informed decisions
  • Help maintain platform quality
  • Build reputation as a trusted mediator

Prerequisites​

Before you start:

  1. Set up a Web3 wallet (MetaMask recommended)
  2. Get test tokens from supported networks
  3. Understand basic blockchain transactions
  4. Have relevant expertise in task domains
  5. Understand dispute resolution principles

Finding Disputes​

Browse Available Cases​

  1. Navigate to the Auditor page

  2. Review available disputes:

    • Task details
    • Dispute reason
    • Involved parties
    • Timeline
    • Required expertise

Evaluating Cases​

Before accepting, consider:

  • Your expertise in the domain
  • Time commitment required
  • Complexity of the dispute
  • Available evidence
  • Potential conflicts of interest

Reviewing Disputes​

Initial Assessment​

  1. Review complete task history
  2. Examine all communications
  3. Study deliverables
  4. Understand dispute context
  5. Identify key issues

Investigation Process​

  1. Review evidence thoroughly
  2. Request additional information if needed
  3. Consider both perspectives
  4. Evaluate against requirements
  5. Document findings

Making Decisions​

Decision Process​

  1. Review all gathered information
  2. Apply relevant standards
  3. Consider platform guidelines
  4. Make objective assessment
  5. Document reasoning

Best Practices​

  • Remain neutral and unbiased
  • Focus on evidence
  • Apply consistent standards
  • Document thoroughly
  • Explain decisions clearly

Communication​

With Parties​

  1. Maintain professional tone
  2. Ask clear questions
  3. Set expectations
  4. Provide updates
  5. Explain decisions

Documentation​

  • Record all interactions
  • Maintain evidence trail
  • Document reasoning
  • Write clear decisions
  • Archive case materials

Managing Multiple Cases​

Case Organization​

  • Track different disputes
  • Manage timelines
  • Document progress
  • Organize evidence
  • Maintain case notes

Time Management​

  • Set review schedules
  • Plan investigation steps
  • Allow time for responses
  • Meet platform deadlines

Building Reputation​

Best Practices​

  1. Handle cases fairly
  2. Meet deadlines
  3. Communicate clearly
  4. Document thoroughly
  5. Maintain professionalism

Professional Growth​

  • Learn from each case
  • Stay updated on standards
  • Expand expertise
  • Build credibility

Handling Complex Cases​

Advanced Techniques​

  1. Break down complex issues
  2. Seek additional expertise
  3. Use structured analysis
  4. Consider precedents
  5. Document thoroughly

Risk Management​

  • Identify potential issues
  • Maintain objectivity
  • Document carefully
  • Seek guidance when needed
  • Follow platform guidelines

Tips for Success​

  1. Thorough Review

    • Examine all evidence
    • Consider all perspectives
    • Document findings
    • Verify facts

  2. Clear Communication

    • Be professional
    • Stay neutral
    • Explain clearly
    • Document everything

  3. Time Management

    • Plan effectively
    • Meet deadlines
    • Allow response time
    • Stay organized

  4. Quality Decisions

    • Be objective
    • Apply standards
    • Explain reasoning
    • Document thoroughly

Getting Help​

  • Join our Discord
  • Connect on Telegram
  • Read documentation
  • Ask community members

Next Steps​

  • Explore Technical Docs
  • Join Community Programs
  • Learn About Smart Contracts

Remember: Start with simpler cases to build experience before taking on more complex disputes.
